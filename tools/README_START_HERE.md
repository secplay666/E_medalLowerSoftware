## 🐛 常见问题快速答案

### Q: 无法连接到MCU
**A:**
1. 检查USB线接好了吗?
2. 串口驱动装了吗? (CH340/PL2303)
3. 试试不同的COM口?
4. 重启浏览器?

### Q: 传输失败/很慢
**A:**
1. 先试试降低波特率到57600
2. 增加超时到5000ms
3. 启用调试模式看日志
4. 检查MCU是否已上传新固件

### Q: 图像显示效果不对
**A:**
1. 调整二值化阈值 (尝试100-150)
2. 启用/关闭线稿提取试试
3. 尝试反色选项
4. 检查目标图像尺寸

### Q: 怎样只传输图像的一部分?
**A:**
1. 勾选"启用剪裁"
2. 设置XY偏移和宽高
3. 应用处理
4. 传输

详见: `V3_升级版功能说明.md` 的故障排查章节
